# lib-imgui

### 简介
* 利用CMake将 [imgui](https://github.com/ocornut/imgui) 作为的一个库，方便快速开发轻量级的GUI应用
* 窗口[glfw](https://www.glfw.org/) + 渲染库[opengl3](https://www.opengl.org/)
* 跨平台，支持linux & Windows
